# Lab Scope

## Purpose

This lab demonstrates basic Windows security event log analysis in a controlled environment. The goal is to understand common Windows event types and document observations in a SOC-style format.

## Environment

The lab is performed on a Windows system owned or controlled by me.

## In Scope

- Windows Security logs
- Failed logon events
- Successful logon events
- PowerShell activity
- Windows Defender events
- Event Viewer screenshots
- SOC-style documentation

## Out of Scope

- Third-party systems
- Production environments
- Unauthorized access attempts
- Malware execution
- Credential theft
- Collection of real user credentials
- Uploading sensitive logs or personal data

## Rules of Engagement

- Only analyze systems owned or controlled in the lab
- Do not upload passwords, private data, or sensitive logs
- Sanitize screenshots if personal information is visible
- Generate only safe test activity
- Document findings clearly and honestly
