# Windows Event Log Analysis Lab

## Overview

This project demonstrates basic Windows security event log analysis in a controlled lab environment. The objective is to generate, review, and document common Windows security events such as failed logons, successful logons, PowerShell activity, and Windows Defender alerts.

## Objectives

- Understand common Windows security event logs
- Analyze failed and successful logon events
- Review PowerShell activity
- Review Windows Defender security events
- Document findings in a SOC-style format
- Build practical Windows log analysis skills for cybersecurity roles

## Lab Environment

| Component | Description |
|---|---|
| Target System | Windows 10/11 VM or local Windows system |
| Tool | Windows Event Viewer |
| Optional Tool | PowerShell |
| Documentation | Markdown and GitHub |

## Planned Event Analysis

| Analysis | Focus |
|---|---|
| Failed Logon Analysis | Review failed authentication events |
| Successful Logon Analysis | Review successful authentication events |
| PowerShell Activity Analysis | Review PowerShell-related event activity |
| Windows Defender Event Analysis | Review antivirus/security-related events |

## Skills Demonstrated

- Windows Event Viewer usage
- Security log analysis
- Authentication event review
- PowerShell activity review
- Windows Defender event review
- SOC-style documentation
- Evidence-based reporting

## Disclaimer

This project is performed in a controlled lab environment for educational and defensive security purposes only. No unauthorized systems or third-party environments are tested.
